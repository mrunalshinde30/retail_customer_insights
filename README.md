# 🛍️ Retail Customer Insights Analysis

An end-to-end Data Analytics project that analyzes retail customer shopping behavior using **Python**, **PostgreSQL**, and **Power BI**. This project demonstrates the complete analytics workflow, from raw data cleaning to business intelligence dashboard creation.

---

## 📌 Project Overview

Retail businesses collect large volumes of customer transaction data every day. Analyzing this data helps organizations understand customer behavior, identify purchasing trends, improve customer experience, and make informed business decisions.

In this project, I performed data cleaning using Python, business analysis using PostgreSQL, and created an interactive Power BI dashboard to visualize customer insights.

---

## 🎯 Business Problem

The objective of this project is to analyze customer shopping behavior and answer important business questions such as:

- Which customer segment generates the highest revenue?
- Do subscribed customers spend more than non-subscribed customers?
- Which products receive the highest customer ratings?
- How effective are discount campaigns?
- Which products perform best within each category?

The insights obtained from this analysis can help businesses improve marketing strategies, customer retention, inventory management, and overall decision-making.
